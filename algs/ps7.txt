Getting the code and setting up

Like all previous psets, fetch the code for this problem set by executing:

git pull git@code.seas.harvard.edu:cs51-psets/cs51-psets.git master

Important: In order to run this problem set's code, you will have to install the OCaml Graphics module. On the appliance, you can do this by running:

curl http://cs51.seas.harvard.edu/graphics.sh | bash
